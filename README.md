# The Funnel Rule

The Funnel Rule is a thinking system that improves decisions by forcing you to consider multiple frames before concluding.

## What this project is

This repository contains:
- A visual explanation of the Funnel Rule
- An interactive tool that lets you test your decision aperture
- A long-term thinking framework I can return to years later

## The Rule (simple version)

The quality of a decision depends on how many frames you open before deciding.

## Why this exists

Most people don’t make bad decisions because they are stupid.
They make bad decisions because they think inside one frame.

This project solves that.
